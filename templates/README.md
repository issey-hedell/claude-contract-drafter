# 契約書テンプレート（ひな形）

このディレクトリに、自社で使用する契約書のひな形を保存してください。

## 保存方法

1. 契約書のひな形ファイル（Word / テキスト / Markdown）をこのディレクトリに配置
2. ファイル名は契約書の種類が分かる名前にする

### ファイル名の例

```
templates/
├── 秘密保持契約書.md
├── 業務委託基本契約書.docx
├── 金銭消費貸借契約書.md
├── 株式譲渡契約書.md
├── 債権譲渡三者間契約書.md
└── ...
```

## 注意事項

- ひな形がない契約書は作成できません（品質を担保するため）
- 新しい種類の契約書が必要な場合は、まずひな形を用意してください
- ひな形の当事者名・日付等は仮の値（[甲]、[日付]等）で構いません
