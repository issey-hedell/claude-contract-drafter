# 契約書テンプレート（ひな形）

## 構成

**会社別 → 契約種類別 → 複数パターン** の3階層で管理。

```
templates/
├── Hedell/                          # 株式会社Hedell Group
│   ├── 秘密保持契約書/
│   │   └── 標準.md                  ✅
│   ├── 新株予約権投資契約書_J-KISS/
│   │   └── コーラル_v2.01.md        ✅
│   ├── 株式投資契約書/
│   ├── 業務委託契約書/
│   ├── 金銭消費貸借契約書/
│   ├── 株式譲渡契約書/
│   ├── 債権譲渡契約書/
│   ├── 売買基本契約書/
│   ├── ライセンス契約書/
│   ├── 雇用契約書/
│   ├── 賃貸借契約書/
│   └── その他/
│       ├── 覚書/
│       └── 解約合意書/
├── Synlai/
│   ├── 開発委託契約書/
│   ├── 基本契約書/
│   ├── 秘密保持契約書/
│   └── その他/
└── README.md
```

## 使い方

1. 該当する **会社フォルダ** → **契約種類フォルダ** にファイルを配置
2. 対応フォーマット: Markdown (.md) / Word (.docx) / テキスト (.txt)
3. 新しい会社はフォルダごと追加

## 注意事項

- ひな形がない契約書は作成できません（品質を担保するため）
- 当事者名・日付等は仮の値（[甲]、[日付]等）で構いません
