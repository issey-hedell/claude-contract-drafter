# 契約書テンプレート（ひな形）

このディレクトリに、自社で使用する契約書のひな形を保存してください。

## 構成

契約書の種類ごとにフォルダを分け、その中に複数パターンを格納できます。

```
templates/
├── 秘密保持契約書/
│   ├── 標準.md              # 通常の二者間NDA
│   ├── 片務型.md            # 一方のみが開示するパターン
│   └── 三者間.md            # 第三者を含むパターン
├── 業務委託契約書/
├── 金銭消費貸借契約書/
├── 株式譲渡契約書/
├── 債権譲渡契約書/
│   ├── 二者間.md
│   └── 三者間.md
├── 売買基本契約書/
├── ライセンス契約書/
├── 雇用契約書/
├── 賃貸借契約書/
└── その他/
    ├── 覚書/
    └── 解約合意書/
```

## ファイルの追加方法

1. 該当するフォルダにファイルを配置（Word / テキスト / Markdown）
2. ファイル名はパターンが分かる名前にする（例: `標準.md`, `簡易版.md`, `三者間.md`）
3. 新しい種類の契約書はフォルダごと追加

## 注意事項

- ひな形がない契約書は作成できません（品質を担保するため）
- 当事者名・日付等は仮の値（[甲]、[日付]等）で構いません
- 原本の .docx を一緒に保存しておくと参照に便利です（.gitignore で除外されます）
